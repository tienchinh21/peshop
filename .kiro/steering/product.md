# Product Overview

PeShop is a Vietnamese e-commerce platform (nền tảng mua sắm trực tuyến) that provides online shopping functionality for both customers and shop owners.

## Core Features

- **Customer Features**: Product browsing, search, cart management, checkout, order tracking, wishlists, product reviews
- **Shop Management**: Shop registration, product management, order processing, promotion campaigns (vouchers, buy X get Y), dashboard analytics
- **Dual API Architecture**: Integrates with both .NET and Java backend services
- **Multi-language**: Vietnamese language support (vi-VN locale)

## User Roles

- **Customers**: Browse products, make purchases, manage orders
- **Shop Owners**: Manage products, process orders, create promotions, view analytics

## Key Business Logic

- Product variants with properties (size, color, etc.)
- Promotion system (vouchers, buy X get Y campaigns)
- Address management with GoShip integration for shipping
- Review and rating system
- Real-time cart and checkout flow
